## 876. Middle of the Linked List

[link]({{page.url}})

### C++
```c++
class Solution {
 public:
  ListNode* middleNode(ListNode* head) {
    ListNode* fast = head;
    ListNode* slow = head;

    while (fast != nullptr && fast->next != nullptr) {
      fast = fast->next->next;
      slow = slow->next;
    }

    return slow;
  }
};
```
